# CodeAlpha - Basic Network Sniffer

## Objective
This project is a basic network sniffer developed using Python and Scapy. It captures live network packets and displays important packet information.

## Features
- Captures live network traffic
- Displays source IP address
- Displays destination IP address
- Detects TCP, UDP, and ICMP protocols
- Real-time packet analysis

## Technologies Used
- Python
- Scapy
- Npcap
- VS Code

## How to Run
1. Install Python
2. Install Scapy:
   python -m pip install scapy
3. Install Npcap
4. Run:
   python sniffer.py

## Output
The program captures packets and displays:
- Source IP
- Destination IP
- Protocol type
